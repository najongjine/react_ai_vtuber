
============================================================

    샘플 모델
    니지이로 마오 (프로 버전)

============================================================

    블렌드 셰이프나 곱셈색, 스크린색의 응용 예시에 대해 배울 수 있는 모델입니다.
    블렌드 셰이프를 사용하여 입과 눈썹, 모발의 흔들림 등에 차분의 형상을 가산했습니다.
    또한 이펙트 등으로 색상이 변화하는 표현에 곱셈색과 스크린색을 사용하고 있습니다.
    VTuber 용도로 사용하기 쉬운 정면용 모델로, 얼굴의 가동 범위도 넓게 제작되었습니다.

    ※표정용 모션과는 병용하지 않는「마법 이펙트와 표정 부착용 모션」도 포함됩니다.

------------------------------
    샘플 데이터 구성
------------------------------

    - 모델 데이터 (.cmo3)
    - 기본모션 (.can3)
    - 삽입용 파일일식 (runtime 폴더)
        - 모델 데이터 (.moc3)
        - 표정 데이터 (.exp3.json)
        - 모션 데이터 (.motion3.json)
        - 모델 설정 파일 (.model3.json)
        - 물리 연산 설정 파일 (.physics3. json)
        - 포즈 설정 파일 (.pose3.json)
        - 표시보조파일 (.cdi3.json)
